## ポートフォリオある？

- [Shopifyアプリ](https://github.com/hltree/shopify-scheduled-delivery)<br>
即動かせるものがなく、申し訳ないのですが・・<br>
Shopify上の注文情報をCSVで出力できるアプリです。おまけとしてカート上に配送日時指定フィールドを生成する機能もあります。<br>
ShopifyデフォルトのCSVとの違いはCSV列を自由にカスタマイズできるところです。<br>
CSVの組み合わせを変えることでヤマト運輸・佐川便など、どの配送業者にも対応できるCSVを作成することができます。
